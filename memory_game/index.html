Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku, Pexeso or simply Paris, is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

Description

This game is helps to improve your memory and keep your mind active! Play it with a friend and see who can complete the game the quickest! This classic game is fun for all ages
